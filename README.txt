git init = Para posibilitar a utilização do GIT 

git status = para mostar os status da pasta exemplo (Se está no commit, se está na linha do tempo master,etc...)

git add = para adicionar o arquivo ao GIT local 

git commit = para criar um commiti local e botar seu nome 

git remote add origin = para adicionar um arquivo a um repositorio remoto(online)

git push -u origin master =  Para adcionar os arquivos no repositorio
